Configuration

  • Database Connection
    • Enter email and password for logging in and set the display data location.
  • Wi-Fi Settings
    • Connect to a Wi-Fi network with either WPA2-PSK (usually home Wi-Fi) or WPA2-Enterprise (usually work or school Wi-Fi).
  • Hosted Access Point Settings
    • Configure the name and password of the hosted access point (i.e. the Wi-Fi network you're connected to right now).
  • Upload New Firmware
    • Manually upload a firmware update to this device.
